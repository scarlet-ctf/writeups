Developer Documentation

Welcome to the JWT Studio API. Use these endpoints to integrate token generation into your own workflows.

Authentication

POST /generate

Returns a signed HS256 JWT. Requires username and password fields in the request body.

System Information

GET /view?page=docs.html

Retrieves static documentation and system status pages.

Security Policy

All tokens are signed using our internal private key. Do not share your SECRET_KEY with anyone. Our internal file viewer is restricted to the static directory for security.
